## 이 서비스가 푸는 문제

축구를 보다 보면 누구나 한 번쯤 "내가 감독이라면 저렇게 안 하는데"라고 생각합니다.
그런데 그 생각을 **확인해 볼 방법이 없습니다.**

웹에는 이미 여러 라인업 빌더가 있지만, 조사한 범위에서 이들은 모두 **배치를 그리고 이미지로 내보내는 도구**입니다.
선수를 옮겼을 때 *그래서 결과가 어떻게 달라지는지*를 알려주지 않습니다.

이 서비스는 거기서 시작합니다. **배치를 바꾸면 예측이 바뀝니다.**

## 3가지 모드

| 모드 | 무엇을 하는가 |
|---|---|
| **전술보드** | 참가국 스쿼드를 드래그앤드롭으로 배치하고 전술 파라미터(라인 높이·압박 강도·템포·폭)를 조정 |
| **리플레이** | 2026 월드컵 실제 경기의 라인업에서 출발해, 다른 선택을 했다면 어땠을지 재시뮬레이션 |
| **경우의 수** | 조별리그 결과 조합을 조작해 토너먼트 진출 확률을 몬테카를로 시뮬레이션으로 산출 |

세 모드를 관통하는 축은 하나입니다 — **어떤 조작이든 즉시 예측치가 갱신된다.**

## 기술적 특징

**학습은 오프라인, 추론은 브라우저에서.**

머신러닝 모델은 Python으로 사전 학습한 뒤 ONNX로 내보내고, 실제 추론은 **브라우저 안에서 100% 로컬로** 수행합니다.

이 설계의 결과:
- **API 키가 필요 없습니다** — 누구나 링크만 열면 전체 기능을 확인할 수 있습니다
- **서버 호출이 없습니다** — 조작에 대한 반응이 즉각적입니다
- **오프라인에서도 동작합니다**

| 모델 | 역할 |
|---|---|
| xG 모델 | 슛 상황 → 득점 확률 |
| 경기 결과 예측 | 팀 강도 → 스코어라인 확률 분포 |
| 포메이션 상성 | 전술 파라미터 조합 → 기대 득점 차 |
| 선수 역할 적합도 | 선수 특성 → 포지션 적합 점수 |
| 몬테카를로 | 조별리그 반복 시뮬 → 진출 확률 |
