# F09. 리플레이 경기 선택

> **우선순위**: P1 · **담당 화면**: S2a · **근거**: P11·P7 · [인덱스](기능명세_인덱스_v1_0.md)

## 1. 개요 (Overview)

- **User Story**: "사용자는 2026 조별리그 한국 3경기 중 하나를 골라 그 경기로 되돌아간다."
- **Why this priority**: 리플레이 모드의 관문. 3경기는 모두에게 동일한 '공통 퍼즐'(P8) — 비교 가능성의 출발점.
- **Independent Test**: 3경기 카드의 표기(상대·개최지·스코어)가 확정 데이터와 일치하는 것만으로 단독 시연 가능.

## 2. 조건 및 상태 (Conditions)

- **Pre-condition**: S1 헤더의 모드 탭에서 리플레이 진입
- **Post-condition**: 선택 경기의 S2b(`/replay/[matchId]`)로 이동, 해당 경기 데이터 로드

## 3. UI/UX·네비게이션

- 3경기 카드: 상대 국기(flag-icons, MIT) + 라운드 + 개최지 + 스코어. 모바일 세로 스택 / 데스크톱 3열
- 진입 카피: "이 경기, 당신이라면?" 톤 `[설계 결정]` — 비하 금지 검수 대상
- 라우팅: `/replay` → 카드 탭 → `/replay/[matchId]`

## 4. 상세 로직·설계 근거 — 확정 데이터 (전부 P11)

| matchId | 상대 | 결과 | 개최지 | 일자 |
|---|---|---|---|---|
| cze | 체코 | **2-1 승** | Estadio Akron, Zapopan(과달라하라) | 현지 6/11 · KST 6/12 |
| mex | 멕시코 | 0-1 패 | Estadio Akron, Zapopan(과달라하라) | — |
| rsa | 남아공 | 0-1 패 | Estadio BBVA, Guadalupe(몬테레이) | — |

- **전 경기 멕시코 개최** — "한국 개최 뉘앙스"는 P11이 정정한 항목이므로 어떤 화면에도 국내 개최 암시 문구 금지
- 체코전 날짜는 현지/KST 병기 (P11)

## 5. 요구사항 (EARS)

| ID | 패턴 | 요구사항 |
|---|---|---|
| F09-R1 | WHEN | **WHEN** 경기 카드를 탭하면, 시스템은 해당 `matchId`의 S2b로 이동하고 타임라인 데이터를 로드한다 |
| F09-R2 | Ubiquitous | 시스템은 카드의 상대·스코어·개최지를 `matches.json`(P11 확정값)에서만 렌더한다(하드코딩 중복 금지) |
| F09-R3 | IF-THEN | **IF** 존재하지 않는 `matchId`로 직접 URL 진입하면, **THEN** 시스템은 S2a(경기 선택)로 안내 리다이렉트한다(404 백지 금지) |
| F09-R4 | IF-THEN | **IF** 경기 데이터 로드가 실패하면, **THEN** 시스템은 카드 목록으로 복귀시키고 재시도 안내를 표시한다 |
| F09-R5 | IF-THEN | **IF** 국기 아이콘 로드가 실패하면, **THEN** 시스템은 국가 코드 텍스트(예: CZE)로 대체 표시한다(빈 아이콘 금지) |

## 6. 시나리오·예외

- **Happy Path**: 모드 탭 → 3카드 확인 → 체코전 선택 → S2b 진입
- **Edge Cases** (물음표 → 해소 R):
  - `/replay/xyz` 같은 잘못된 URL로 들어오면? → 선택 화면으로 안내 (F09-R3)
  - 정적 JSON 파싱이 깨지면? → 목록 복귀+재시도 (F09-R4)
  - 국기 에셋이 없으면? → 코드 텍스트 폴백 (F09-R5)

## 7. 수용 기준 (Given-When-Then)

- **Given** S2a **Then** 3카드의 결과·개최지가 P11 확정표와 1:1 일치한다
- **Given** 임의 카드 **When** 탭 **Then** 해당 경기 S2b로 이동한다
- **Given** 잘못된 matchId URL **When** 진입 **Then** 백지·에러 화면 없이 S2a로 안내된다

## 8. 데이터·상태·비기능

- **데이터**: `src/data/matches.json` — P11 확정 3경기 메타(개최지·스코어·일자)
- **비기능**: 카드 터치 타겟 ≥1cm · flag-icons(MIT) 라이선스 고지 대상 (P7) · 카피 비하 금지 검수
