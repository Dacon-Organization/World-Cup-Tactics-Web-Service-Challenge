## 표기 규약 (전 문서 공통)

- `(Pn)` — [리서치 원장](../../../research/RESEARCH-LOG.md) `회수됨` 항목으로 역추적
- `ADR-n` — 리서치로 확정되지 않는 설계 재량 결정 ([decisions/](../../decisions/))
- `[설계 결정]` — 파일 내 국소 설계 재량 (ADR 승격 전 단계)
- `[추정 설계목표]` — 구현 후 실측 검증 항목 (QA 체크리스트 이관)
- `[NEEDS CLARIFICATION: 질문]` — 미해소 사항. **구현 착수 전 전량 해소가 게이트** (spec-kit)
- 요구사항 문장은 **EARS 표기법**: WHEN(이벤트)/WHILE(상태)/WHERE(선택)/**IF-THEN(오류·이상)** — IF-THEN 기능당 ≥3 의무
- 수용 기준은 **Given-When-Then** — 실기기에서 검증 가능한 문장만
- 선수 표기는 완전 추상(FW-A, MF-B…) — 가공명 확정은 데이터 설계에서 (P7)
