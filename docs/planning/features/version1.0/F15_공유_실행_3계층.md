# F15. 공유 실행 3계층

> **우선순위**: P2 · **담당 화면**: S1·S2b · **근거**: P8·P12 · [인덱스](기능명세_인덱스_v1_0.md)

## 1. 개요 (Overview)

- **User Story**: "사용자는 자기 환경에서 가장 자연스러운 방법(공유 시트/URL/이모지 텍스트)으로 전술을 공유한다."
- **Why this priority**: 환경별 폴백이 없으면 공유가 '어떤 브라우저에선 안 되는 기능'이 된다 — 완성도 25점과 1차 투표 둘 다에 직결.
- **Independent Test**: Web Share 지원/미지원 브라우저 각각에서 공유 버튼 1개가 올바른 계층으로 동작하는 것만으로 단독 시연 가능.

## 2. 조건 및 상태 (Conditions)

- **Pre-condition**: 공유할 상태 존재, URL 생성 가능(F13)
- **Post-condition**: URL 공유 완료(시트/클립보드) 또는 이모지 그리드 클립보드 복사 완료 + 완료 피드백 표시

## 3. UI/UX·네비게이션

- 공유 버튼 1개 → 환경 감지로 최적 경로 자동 선택 + 보조 옵션("텍스트로 복사") 제공
- 계층: ① URL 복사(전 환경 기본) ② **Web Share API** — OS 네이티브 시트 ③ **클립보드 이모지 그리드**
- 이모지 그리드: F07 10×10 배열의 텍스트 번역(예: `🟩×6 🟨×2 ⬜×2` 행렬) — 스코어 스포일러 없이 확률 구조만 (P8 스포일러 프리)

## 4. 상세 로직·설계 근거

- `navigator.share` **feature detection 필수** — Firefox 데스크톱 미지원 (P8). HTTPS + 사용자 제스처(transient activation) 요건
- `navigator.clipboard.writeText` 폴백, 그마저 실패 시 선택 가능한 텍스트 노출
- 이모지 그리드 = F07 배열과 동일 데이터의 텍스트 표현 (P8·P12 정합 — 동일 포맷 3표현 원칙)
- **설계 근거**: Wordle 이모지 그리드의 플랫폼 독립 텍스트 원리 (P8 1차 출처)

## 5. 요구사항 (EARS)

| ID | 패턴 | 요구사항 |
|---|---|---|
| F15-R1 | WHERE | **WHERE** `navigator.share`가 존재하면, 시스템은 공유 버튼으로 OS 공유 시트를 연다(사용자 제스처 내에서만) |
| F15-R2 | WHEN | **WHEN** 이모지 그리드 복사가 실행되면, 시스템은 스코어 없이 확률 구조와 경기 식별 텍스트만 담은 그리드를 클립보드에 넣는다 |
| F15-R3 | WHEN | **WHEN** 어떤 계층이든 공유가 완료되면, 시스템은 완료 피드백(토스트)을 표시한다 |
| F15-R4 | IF-THEN | **IF** `navigator.share`가 없거나 호출이 거부되면, **THEN** 시스템은 URL 클립보드 복사로 자동 폴백하고 그 사실을 피드백에 표시한다 |
| F15-R5 | IF-THEN | **IF** 사용자가 OS 공유 시트를 취소하면, **THEN** 시스템은 이를 오류로 표시하지 않고 조용히 원상태로 복귀한다 |
| F15-R6 | IF-THEN | **IF** 클립보드 접근이 거부되면(권한·비보안 컨텍스트), **THEN** 시스템은 URL·텍스트를 선택 가능한 상태로 화면에 노출해 수동 복사를 안내한다 |

## 6. 시나리오·예외

- **Happy Path (모바일)**: 공유 버튼 → OS 시트 → 카톡 선택 → 전송(미리보기는 F14)
- **Happy Path (데스크톱 Firefox)**: 공유 버튼 → URL 클립보드 복사 + 안내
- **Edge Cases** (물음표 → 해소 R):
  - 공유 시트를 열었다 그냥 닫으면 에러? → 정상 취소 처리 (F15-R5)
  - 클립보드 권한을 막아놨으면? → 수동 복사 UI (F15-R6)
  - 이모지 그리드에 스코어가 새면? → 구조상 확률만 포함 (F15-R2)

## 7. 수용 기준 (Given-When-Then)

- **Given** iOS/Android **When** 공유 버튼 **Then** OS 공유 시트가 열린다
- **Given** Firefox 데스크톱 **When** 공유 버튼 **Then** 크래시 없이 URL이 클립보드에 복사되고 안내가 뜬다
- **Given** 이모지 그리드 복사 **When** 아무 곳에 붙여넣기 **Then** 확률 구조+경기 식별자가 텍스트로 붙고 스코어는 없다

## 8. 데이터·상태·비기능

- **연계**: URL은 F13, 카드 이미지는 F14, 그리드 데이터는 F07 — 본 기능은 배포 계층만 담당(생성 로직 없음)
- **비기능**: HTTPS 전제(Vercel) · 사용자 제스처 요건 준수 · 이모지 그리드 카피도 비하 금지 검수 대상
