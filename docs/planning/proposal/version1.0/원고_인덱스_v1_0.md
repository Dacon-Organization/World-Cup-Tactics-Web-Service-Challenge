## 조판 인계 규약

1. **`[조판: ]` 태그**: 각 절 말미의 태그가 그래픽 제작·배치 지시다. mermaid 코드블록은
   렌더 후 이미지로 삽입, 표는 스타일 적용 후 그대로
2. **(Pn) 표기**: 조판 시 각주 번호로 변환하고 16절 매핑 표와 연결
3. **"검수 메모 (조판 제외)" 섹션**: PDF에 싣지 않는다 — 원고 품질 관리용 메타데이터
4. **서비스명**: "만약의 감독" (1절 결정 기록) — 전 절 표기 통일 확인됨
5. **조판 후 최종 검수**: [기획서-골격.md](../../기획서-골격.md) 말미의 "최종 검토
   체크리스트" 7항을 PDF 상태에서 재실행 (비하·실명·FIFA IP는 그래픽 포함 전수)
