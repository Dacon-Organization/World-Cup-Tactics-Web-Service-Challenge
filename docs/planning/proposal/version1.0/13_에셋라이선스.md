# 13. 디자인 시스템·에셋/라이선스 전략

> 분량 목표: 1p · 방어: **실격 방어** (공정성/저작권) · 근거: P7·P10

---

**핵심 메시지 — 실격 리스크를 0으로: 모든 에셋의 라이선스를 표로 증명한다.**

저작권·라이선스 위반은 실격 또는 감점, 비하 콘텐츠는 심사 제외다. 이 절은 심사자가 물을
법한 질문 — *"선수 실명을 써도 되나요?"* — 에 대한 선제 답변이며, 방어가 아니라 설계
역량으로 서술한다.

## 선수 표기 — 완전 가공명, 법리로 선을 그었다

한국 법원은 성명을 **변형해도 특정인이 연상되면** 퍼블리시티권 침해로 판단한 바 있다
(서울서부지법 2010카합245, 원문 확인, P7). 게임 업계의 연상형 변형명 관행은 이 법리와
저촉 소지가 있어 참고하지 않았다. 따라서 사용자에게 노출되는 모든 표면(UI·공유 이미지·
클라이언트 JSON·이 기획서 포함)에서 실명·이니셜·연상형 변형명을 **3단계 전면 배제**하고,
음운 규칙 기반의 **완전 창작 가공명**만 사용한다 (P7). 가공명 배정은 무작위이며(역추적
가능한 변환 규칙 자체를 금지), 빌드 검증 스크립트가 실명·유사명을 기계 검사한다.

이는 주최 측의 "더미 데이터 직접 구성 권장" 안내와 정합하고, 학습층 실데이터(9절)와의
경계는 스키마 수준에서 유지된다.

## 대회 상징 — FIFA 공식 IP 일절 배제

엠블럼·워드마크·트로피·마스코트·공식 서체를 일절 쓰지 않는다. "World Cup™" 단독 표기까지
등록 워드마크임을 원문에서 확인했다 (P7). 대체 전략의 근거는 FIFA 가이드라인 자체에 있다 —
FIFA는 비권리자에게 "generic football or country-related images and/or terminology"
사용을 권장한다 (FIFA IP Guidelines §5, 원문 p12, P7). 그래서 이 서비스의 시각 자산은
**국기 아이콘 + 팀 대표색 팔레트 + 자체 제작 SVG(피치·전술 기호)** 로만 구성한다.
같은 가이드라인 §7이 "선수 등 제3자 권리는 별도"임을 명시하므로 (P7), 위 가공명 원칙이
독립적으로 필요하다는 점도 원문으로 뒷받침된다.

## 카피 규율 — 비하 금지는 문장 단위로

특정 선수·감독·팀·국가의 판단을 "잘못·실수·패착"으로 규정하는 표현을 전 화면·전 카드·전
문서에서 금지하고, 조판 단계에서 카피 전수 검수를 거친다. 서술 관점은 하나다 —
**"다른 선택지가 있었다."**

재사용해 "실명은 파이프라인 안, 화면에는 가공명" 한 컷. 캡션 "모든 에셋에 출처와 라이선스가 있다 (P7·P10)"]`
