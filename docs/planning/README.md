## 원칙

### 모든 수치는 역추적 가능해야 한다

기획서에 들어가는 수치는 `../research/RESEARCH-LOG.md`의 `반영됨` 항목으로 역추적되어야 합니다.
**역추적이 안 되는 수치는 기획서에서 뺍니다.** 출처 없는 숫자는 설득력이 아니라 리스크입니다.

### 비하 표현 금지

특정 선수·감독·팀·국가를 폄훼하는 표현은 **심사 제외 사유**입니다.
"누가 잘못했는가"가 아니라 "어떤 선택지가 있었는가"로 서술합니다.
최종 검토 시 전수 확인합니다.

### 불확실성을 숨기지 않는다

예측 모델의 한계를 기획서에서 먼저 밝힙니다.
확률을 단정적 예측처럼 포장하는 것이 진짜 리스크이며,
한계를 정직하게 다루는 것이 오히려 기획의 신뢰도를 높입니다.
