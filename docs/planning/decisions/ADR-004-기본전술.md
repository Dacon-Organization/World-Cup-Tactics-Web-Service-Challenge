# ADR-004. 첫 진입 기본 전술 — 4-3-3 + 슬라이더 중앙값

- **Status**: Accepted (2026-07-22)
- **관련**: [F08](../features/version1.0/F08_온보딩_기본상태.md), P5

## Context

P5는 "첫 진입 시 기본값으로 이미 예측치 표시(빈 화면 금지)"를 확정했지만,
그 기본값이 무엇인지는 설계 재량이다.

## Decision

**4-3-3 포메이션 + 슬라이더 4종 전부 중앙값(보통)** 을 기본 상태로 확정한다.

- 4-3-3: 대회요강 원문 예시 첫 번째 포메이션이자 가장 인지도 높은 공격 포메이션
- 슬라이더 중앙값: 어느 방향으로든 조작 여지가 대칭적 — "움직여 보고 싶다"는 유도가 최대
- 기본 상대는 리플레이 1차전 상대(체코)로 설정 `[NEEDS CLARIFICATION: 기본 상대를 체코로 할지, 중립 가상 팀으로 할지 — 데이터 설계 시 가공명 체계와 함께 확정]`

## Consequences

- (+) 심사자가 진입 1~2초 내 "조작→반응" 구조 발견 (P5 온보딩 목표)
- (+) 중앙값 시작이므로 어떤 조작이든 확률 변화가 관찰됨 — 데드존 없음
- (−) 기본 상대 미확정 1건이 데이터 설계로 이월됨 (NEEDS CLARIFICATION 게이트)
