# ADR-007. 검증 분할 — 대회 단위 시간 분할 (train ≤2014 / valid 2018 / test 2022)

- **Status**: Accepted (2026-07-23)
- **관련**: [평가_설계_v1_0.md](../ml/version1.0/평가_설계_v1_0.md) §3 · P12 · ADR-006

## Context

성능 방어선(RPS 0.19~0.22 / 정확도 0.50~0.55)의 출처는 2018 WC(64경기)+2022 WC(64경기)
합계 128경기를 **out-of-sample로 측정한** 공개 실측 baseline이다(P12). 우리 모델의 수치를
이 방어선과 비교하려면 **측정 조건(어떤 경기를 학습에서 제외하고 평가했는가)이 같아야**
비교가 유효하다 — 조건이 다르면 "방어선 충족" 주장 자체가 심사자 반박(T5)에 취약해진다.

또한 축구 경기 데이터는 시계열이다. 무작위 K-fold는 미래 경기가 학습에, 과거 경기가 검증에
섞여 **Elo·폼 같은 누적 피처를 통해 미래 정보가 새는(리키지)** 구조적 결함을 만든다.

## Decision

**대회 단위 시간 분할을 채택한다.**

| 구간 | 대회 | 용도 |
|---|---|---|
| **train** | 남자 WC 1930~2014 (20개 대회) | 모델 학습 |
| **valid** | 남자 WC 2018 (64경기) | Optuna 탐색 · 앙상블 가중 산출 · 모델 선택 |
| **test** | 남자 WC 2022 (64경기) | **최종 평가 1회 — 방어선 대조 전용** |

1. test(2022)는 하이퍼파라미터 탐색·모델 선택에 일절 사용하지 않고, 전 파이프라인 확정 후
   **한 번만** 평가한다. 여러 번 보면 사실상의 검증 셋이 되어 방어선 대조가 무효화된다.
2. 리키지 방지: Elo·폼 등 누적 피처는 **해당 경기 킥오프 이전 정보만**으로 계산한다(rolling).
   전처리 통계(스케일링 등)도 train에서만 적합(fit)한다.
3. 여자 WC(1991~2019) 증강·전 데이터 재학습(refit)은 **실험 항목**: valid 성능이 개선될 때만
   채택하고, 어떤 경우든 **보고하는 테스트 수치는 위 분할 기준**으로 산출한다.

## Consequences

- (+) 방어선 출처와 측정 조건이 정합 — "우리 RPS 0.21 vs 방어선 0.19~0.22"가 같은 자(尺)의
  비교가 되어 기획서 10절 주장의 방어력이 생김
- (+) 시간 분할이라 Elo·폼 리키지가 구조적으로 차단됨 — 검증 수치의 신뢰성
- (−) valid가 단일 대회 64경기로 작음 → Optuna가 2018 특성에 과적합할 위험.
  완화: 탐색 시 train 내부 시간순 서브분할(예: ≤2010 / 2014)을 보조 신호로 병용 `[설계 결정]`
- (−) 2022 이후 데이터(2026 조별리그 표층)는 평가에 쓸 수 없음 — 2026 3경기는 표본이 아니라
  **입력**(리플레이 컨텍스트·Elo 연장)으로만 사용한다는 경계가 명확해짐 (피처_정의서 §3)
