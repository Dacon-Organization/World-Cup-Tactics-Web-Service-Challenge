# ADR-002. 전술 슬라이더 4종 구성 — 라인 높이·압박 강도·공격 폭·템포

- **Status**: Accepted (2026-07-22)
- **관련**: [F04](../features/version1.0/F04_전술_슬라이더.md), P5

## Context

P5(UX 리서치)는 "이분법 토글보다 연속 슬라이더가 감독 행위감을 살린다"와 라벨 예시로
"라인 높이/압박 강도"를 확정했고, 1차 노출 슬라이더 수를 4개로 제시했다.
그러나 나머지 2개 슬라이더가 무엇이어야 하는지는 리서치로 확정되지 않았다(근거 공백이 아니라 설계 재량).

## Decision

**라인 높이 · 압박 강도 (P5 예시 채택) + 공격 폭 · 템포 (설계 재량)** 4종으로 확정한다.

- 공격 폭·템포 선정 기준: ① 축구 중계·전술 담론에서 통용되는 감독 용어일 것 ② 피치 위 시각 프리뷰로 즉시 표현 가능할 것(폭→좌우 산개, 템포→화살표 밀도) ③ 모델 입력 피처로 매핑 가능할 것
- 모든 슬라이더는 감독 용어 + "낮음/보통/높음" 질적 라벨 병기 (P5)

## Consequences

- (+) 4종 모두 시각 프리뷰 100ms 요건(P5)을 만족하는 표현이 존재
- (−) 공격 폭·템포의 모델 기여도는 학습 단계에서 검증 필요 — 기여도가 낮으면 해당 슬라이더의 예측 반영 방식을 조정해야 함 (ML 설계 단계에서 확인, 기획서에는 "설계 결정"으로 표기)
