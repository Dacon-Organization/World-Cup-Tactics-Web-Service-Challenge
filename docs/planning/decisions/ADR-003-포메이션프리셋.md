# ADR-003. 포메이션 프리셋 4종 — 4-3-3 / 4-4-2 / 3-4-3 / 3-5-2

- **Status**: Accepted (2026-07-22)
- **관련**: [F01](../features/version1.0/F01_전술보드_피치.md)

## Context

전술보드는 포메이션 프리셋에서 시작해 자유 배치로 이어진다. 프리셋 수와 종류는
리서치 확정 사항이 아닌 설계 재량. 대회요강 대회 소개가 "4-3-3이 아니라 3-4-3으로 가야지"를
직접 예시로 들고 있다.

## Decision

**4-3-3 / 4-4-2 / 3-4-3 / 3-5-2** 4종으로 확정한다.

- 4-3-3·3-4-3: 대회요강 원문이 직접 언급한 두 포메이션 — 심사자 기대와 일치
- 4-4-2: 가장 보편적인 참조 포메이션
- 3-5-2: 백3 계열 변형으로 전술 다양성 확보
- 4종 초과는 첫 화면 선택 부담(점진적 공개 원칙, P5)과 프리셋별 QA 비용을 고려해 제외

## Consequences

- (+) 프리셋별 표준 좌표 4세트만 QA하면 됨 — 완성도 관리 용이
- (+) 자유 배치(F02)가 있으므로 프리셋 수가 적어도 표현력 손실 없음
- (−) 5백·다이아몬드 등 마니아 포메이션 미제공 — 자유 배치로 우회 가능함을 기획서에 명시
