## 요약 및 판단

### 채택한 결론

**① 성능 방어선(기획서 인용 가능)**: 예측 성능 목표 **RPS ≈ 0.19~0.22 / 정확도 ≈ 0.50~0.55**.
- 근거: 국제대회 128경기 실측 baseline(RPS 0.209~0.219, 정확도 0.508~0.547, [arXiv 2606.24171](https://arxiv.org/pdf/2606.24171v1.pdf))과 538 클럽 RPS 0.1957이 수렴.
- ⚠️ **arXiv의 RPS 0.127(고급 모델)은 128경기 소표본 재적합 값 → 벤치마크로 인용 금지**(심사자 반박 T5 취약). 격하해 다룸.

**② 시뮬레이션 반복 횟수**: **정밀 25,000회 / 빠른 5,000회**.
- 근거: 538 클럽 20,000·Opta 25,000·arXiv 진출확률 5,000. ±1%p 밴드 목표 시 N≥약 10,000(표본크기 공식).
- P3의 "빠른 1,000 / 정밀 10,000" 제안을 **상향 조정**(공개 사례 정합성 ↑). → 원장 반영 시 조정 근거 명기.

**③ 신뢰구간 = Wilson score 채택**: 진출확률 0%/100% 근접 팀에 Wald 대신 필수(항상 [0,1] 내부). 순수 JS 즉시 계산(원문 TS 스케치). SE = √(p̂(1−p̂)/N).

**④ 불확실성 시각화 역할 분담** [추정, P5 후속]:
- 단일 진출확률 → **아이콘 배열 10×10**(top/row 채움, central/random 편향 배치 회피, 퍼센트 숫자 병기, 스크린리더 텍스트 라벨).
- 여러 시나리오 대조(진출/탈락/조1위) → **HOPs 400ms/프레임**(사전 생성 프레임 재생, <100ms 금지). 각 프레임 = 몬테카를로 1회.
- 상호작용 흐름: 슬라이더 즉시 프리뷰(100ms) → debounce 후 시뮬레이션(정밀은 진행률·취소 노출) → Wilson 밴드 표시. HOPs는 추론과 분리(사전 렌더).

**⑤ 핵심 모델(P3 정합)**: Elo 차이 covariate 로지스틱/더블포아송 baseline(RPS 0.21대) + Dixon-Coles τ 저점수 보정 + Karlis-Ntzoufras `bivpois` 스코어라인. 모두 Python 학습→ONNX 용이.
