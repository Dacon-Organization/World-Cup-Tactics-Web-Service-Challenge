## 원문을 편집하지 않는 이유

기획서의 각주 출처로 **역추적**하기 위함입니다.
요약하는 과정에서 뉘앙스가 사라지거나 조건이 탈락하면, 나중에 그 수치가 어떤 조건 하에서
유효했는지 알 수 없게 됩니다.

요약이 필요하면 원문 **아래에** 별도 섹션으로 덧붙입니다.

## 회수 후 할 일

1. 이 폴더에 원문 저장
2. `../RESEARCH-LOG.md`의 해당 항목 상태를 `발주됨` → `회수됨`으로 갱신
3. 답변에서 새로 생긴 질문이 있으면 원장에 `대기`로 적재
4. 기존 결론과 모순되는 내용이 있으면 원장에 모순 사실 기록
