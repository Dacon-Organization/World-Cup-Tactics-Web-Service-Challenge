## 요약 및 판단

### 채택한 결론 — 에셋 제작 가이드 확정

**① 선수 실명·이니셜·변형명 전면 회피 (고위험 확정)**
- 서울서부지법 2010. 4. 21.자 **2010카합245** 결정(마구마구 사건): 성명 그대로는 물론, **변형·이니셜도 "특정인 연상" 시 퍼블리시티권 침해**. 부정경쟁방지법 §2 1호(성명·초상 등 무단 사용)와 결합해 한국 관할에서는 안전지대 없음.
- 미국 C.B.C. 판례(공개 통계+실명 허용)에 기대지 않음 — 한국 서비스에는 속지주의상 한국법 우선.
- **채택: 완전 가공명 더미 데이터** (대회요강 171~172행의 "더미 데이터 직접 구성 권장, JSON 구성 가능"과 정합).
- PES식 연상형 변형명("Naldorinho"류)은 위 판례 법리와 저촉 소지 → **참고 관행으로 기각**.

**② FIFA 공식 IP 일절 금지**
- 엠블럼·워드마크·트로피·마스코트·공식 서체 사용 금지. 워드마크 목록에 **"World Cup™" 단독 표기도 등록**되어 있음(원문 PDF p7 직접 확인) → 서비스명·로고에 "월드컵"류 영문 워드마크 편입 금지.
- "2026 월드컵" **서술적 언급은 조건부 허용**: 1회성 본문 서술 + 공식 마크 미사용 + 반복·정적 배치 금지 + "비공식·팬/학습 목적, FIFA와 무관" 고지.

**③ 안전 대체 에셋 확정**
- 등번호 단독 원판, 범용 실루엣, 팀 대표색 팔레트(엠블럼·문양 결합 금지), 식별용 국기 아이콘, 자체 SVG 전술기호·추상 트로피.
- 국기: Paris Convention Article 6ter는 "상표로서의 등록·사용"만 규율(저작권 미창설) → 식별용 표시 저위험.

**④ 폰트·아이콘 채택안**
- 웹폰트: **Pretendard** (SIL OFL 1.1, WOFF2 서브셋 임베딩 허용, 페이지 표시 시 표기 불요·파일 재배포 시 LICENSE 동봉).
- UI 아이콘: **Lucide** (ISC + Feather 유래 일부 MIT). 대체: Heroicons/Tabler/Phosphor (전부 MIT).
- 국기: **flag-icons** (MIT). twemoji는 그래픽 CC-BY 4.0 표기 조건부라 차선.

### P10과의 경계 확정 (정합성 점검 — 원장 기록 대상)

- 원문의 "P10 부재" 서술은 **Space 측 한계**(로컬 원장에는 P10 `회수됨` 존재). 정합성 점검은 여기서 로컬 수행.
- **모순 아님 — 레이어 분리로 통합**: P10이 채택한 `joshfjelstul/world-cup-database`(실선수명 포함, CC BY-SA 4.0)는 **모델 학습·집계 입력 레이어**까지만 사용(개별 사실 데이터는 3개 관할 모두 미보호 — P10 ④). P7 결론은 **사용자 노출 표면 레이어**의 실명 금지.
- **경계 규칙(신규 확정)**: 데이터셋의 실명 컬럼은 학습 파이프라인 내부까지만. 화면·공유 이미지·클라이언트 전달 JSON 등 **사용자 노출 표면에는 완전 가공명 매핑을 거친다.** joshfjelstul 선수명을 화면에 그대로 렌더링하면 P7 위반.
- P2와도 모순 없음(P2=데이터 소스 축, P7=에셋·표현 축). 대회요강 "외부 자산 라이선스 확인" 원칙과 정합.
