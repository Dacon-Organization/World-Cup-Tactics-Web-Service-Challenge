## 요약 및 판단

### 채택한 결론
- **M1 (xG)**: **로지스틱 회귀 소형 모델** 채택. XGBoost 대비 ONNX 크기·속도 이점이 크지 않음(P4와 정합). 핵심 피처 = 슛 거리·각도·수비 압박.
- **M2 (스코어라인 분포)**: Dixon-Coles 포아송이 표준이나 국가대표팀은 표본 부족(A매치 연 8~12경기, 월드컵 조별 3경기). → **공개 Elo/FIFA 랭킹 강도지수를 입력으로 쓰고 하이퍼파라미터(홈 어드밴티지, ρ)만 소규모 보정하는 하이브리드**. [추정, 설계 제안]
- **포메이션 상성 → 승패 직접 연결**: 재현 가능한 학술 연구 **부재(공백 확인)**. → **M4(포지션 적합도)는 학술 근거 대신 자체 휴리스틱(포지션 간 거리 기반 페널티)으로 설계하고 [추정] 명시.**
- **몬테카를로**: 진출 확률은 1,000~5,000회로 안정, 우승 등 희소 이벤트는 10,000회+. **"빠른 모드(1,000)/정밀 모드(10,000)" 선택 설계.** 브라우저 JS 10,000회는 수백 ms 내 처리 가능.
