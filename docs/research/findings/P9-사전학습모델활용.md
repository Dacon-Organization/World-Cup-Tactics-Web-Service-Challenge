## 요약 및 판단

### 채택한 결론 — **노선 (A) 직접 학습 채택**
1. **축구 특화 모델 생태계는 얕다**: HF Hub 축구 결과 예측 모델 대부분 개인 취미 수준·독립 검증 없음. 가장 나은 sergiopesch/wc2026-match-predictor(499경기, CV 0.545)도 "조악한 3피처" 한계를 스스로 인정.
2. **TabPFN은 성능 매력적이나 라이선스·구조가 문제**: 수백~수천 행에서 GBDT 대비 우위가 독립 벤치마크로 확인되나, **TabPFN-2.5는 상업/프로덕션 전면 금지 비상업 라이선스**, in-context learning 구조상 ONNX 고정 그래프 변환에 부적합 가능성.
3. **직접 학습이 라이선스·배포 리스크가 명확히 낮다**: 소형 GBDT는 라이선스를 전적으로 자기 통제 가능, ONNX 변환 후 수백KB~수MB로 브라우저 실행 확실.
- **핵심 원칙**: **"파인튜닝하면 라이선스가 리셋된다"는 통념은 거짓.** CC-BY-NC·커스텀 비상업 조건은 파생물에 그대로 승계. 학습 데이터·베이스 모델 중 하나라도 비상업이면 최종 파생물도 그 조건을 벗어날 수 없음. → **라이선스는 파인튜닝으로 세탁되지 않는다**(원장 개정 이력의 판단과 일치).
- **절충안**: Chronos-Bolt-tiny(9M, Apache-2.0)·Moirai-1.0-R-small(13.8M, Apache-2.0) 같은 완전개방 소형 시계열 모델을 팀 폼 추이 등 **보조 피처 추출**에만 실험적으로. 핵심 예측은 직접 학습 GBDT. 단 실제 ONNX 변환 검증 후에만.
