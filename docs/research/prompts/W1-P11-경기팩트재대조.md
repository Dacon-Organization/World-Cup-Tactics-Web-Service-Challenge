# P11 — 한국 3경기 교체·세부 통계의 FIFA 공식 매치리포트 재대조

| 항목 | 내용 |
|---|---|
| **웨이브** | W1 (심화) |
| **트리거** | T1 근거 공백 |
| **블로킹 대상** | 리플레이 모드 타임라인의 분(分) 단위 정확도, 세부 통계 표시값 |
| **선행 리서치** | [P6](../findings/P6-한국3경기팩트.md) 후속 |

> **왜 지금 필요한가**: P6는 스코어·득점자·선발 라인업·점유율·xG를 FIFA·FotMob·국내기사 3중 교차로 확정했지만,
> **교체 투입 정확한 분(分)이 출처 간 소폭 상이**하고, **코너킥·파울·경고 누적은 FIFA 세부 통계 미접근으로 미확인**,
> **남아공전 xG는 FotMob 내부에서도 불일치(1.16–0.90 vs 1.1–1.0)**로 남았습니다.
> 리플레이 모드는 타임라인을 분 단위로 재생하므로, 서비스에 싣기 전 공식 1차 출처로 이 값들을 확정해야 합니다.

---

## 프롬프트 (복사해서 사용)

```
[역할]
당신은 축구 경기 기록을 1차 출처로 검증하는 스포츠 데이터 팩트체커다.

[배경]
이 Space의 프로젝트 배경을 전제로 한다. 2026 월드컵 대한민국 조별리그(A조) 3경기를
"리플레이 모드"로 분(分) 단위 타임라인으로 재생하는 웹서비스를 만든다.
이전 리서치(P6)에서 스코어·득점자·선발 라인업·점유율·xG는 FIFA 매치센터·FotMob·국내 언론 3중 교차로 확정했으나,
아래 항목은 출처 간 불일치 또는 미접근으로 확정하지 못했다. 이번에 FIFA 공식 1차 출처로 확정하려 한다.
대상 경기:
- 6/12(KST) 대한민국 2–1 체코
- 6/19(KST) 멕시코 1–0 대한민국
- 6/25(KST) 남아프리카공화국 1–0 대한민국

[질문]
1. 세 경기 각각에 대해, FIFA 공식 매치리포트(공식 Match Report PDF 또는 inside.fifa.com/fifa.com
   매치센터의 확정 기록)에서 모든 교체 투입의 정확한 분(分)과 선수(IN/OUT)를 확인하라.
   그 값을 FotMob·국내 기사 표기와 대조해 불일치를 명시하라.
2. 세 경기 각각의 코너킥·파울·경고(옐로/레드) 누적 수치를 FIFA 공식 통계로 확인하라.
   FIFA에서 공개되지 않으면 "미확인"으로 남기고, 대체 1차 출처(Opta 계열 The Analyst 등)가 있으면 함께 제시하라.
3. 남아공전 xG의 상충(FotMob 매치페이지 1.16–0.90 vs FotMob 뉴스기사 1.1–1.0)에 대해,
   가능한 1차/공식에 가까운 출처의 값을 확인하고 어느 값을 채택해야 하는지 판단하라.
4. 세 경기 각각의 정확한 킥오프 시각(KST와 UTC 병기)과 경기장을 FIFA 공식 기록으로 확정하라.

[산출물 형식]
질문 1~4를 다음 컬럼의 표로 답하라(경기별로 구분):
| 경기 | 항목 | FIFA 공식값 | FotMob값 | 국내기사값 | 채택값 | 출처 URL |

표 이후에, 리플레이 모드 타임라인에 그대로 넣어도 되는 "확정 이벤트 목록"(분, 이벤트, 선수)을
경기별로 정리하고, 확정 못한 항목은 별도로 "미확인 목록"에 남겨라.

[출처 요구]
- FIFA 공식 매치리포트/매치센터를 1차 출처로 우선하라. 접근 URL을 반드시 명시하라.
- 값이 출처마다 다르면 모두 병기하고 채택 근거를 밝혀라.
- 확인 불가한 항목은 "미확인"으로 남겨라.

[최신성]
- 대회는 2026-07 종료. 경기 종료 후 확정된 공식 기록을 기준으로 하라(경기 전 예상 라인업 아님).

[배제]
- 특정 선수·감독의 판단을 "실수"·"잘못"으로 규정하는 서술을 배제하라.
  사실(누가 언제 교체되었는가, 통계값)만 기록하라.
- [사실]과 [추정]을 라벨링하라.
- 이 Space의 이전 리서치(P6)와 모순되는 값이 있으면 명시적으로 지적하라.

[언어]
한국어. 선수명은 한글, 필요 시 로마자 병기.
```

## 회수 후 확인 사항

- [ ] 세 경기 교체 투입의 **분(分)**이 FIFA 공식값으로 확정되었는가
- [ ] 코너킥·파울·경고가 확정 또는 명확히 "미확인"으로 분류되었는가
- [ ] 남아공전 xG 채택값이 근거와 함께 결정되었는가
- [ ] 리플레이에 바로 넣을 "확정 이벤트 목록"이 경기별로 정리되었는가

## 이 결과가 바꿀 수 있는 것

| 결과 | 후속 영향 |
|---|---|
| 교체 분이 FIFA 공식값으로 **확정** | 리플레이 타임라인 데이터 스키마에 확정값 입력 |
| 세부 통계가 **미확인** 유지 | 해당 항목은 리플레이 UI에서 표시하지 않거나 "비공식" 라벨 처리 |
| 남아공전 xG 채택값 결정 | 경기 요약 화면 xG 표시값 확정 |
