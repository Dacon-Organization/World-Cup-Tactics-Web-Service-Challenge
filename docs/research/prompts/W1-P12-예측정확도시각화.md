# P12 — 예측 모델 국가대표 정확도 및 확률 시각화 스펙

| 항목 | 내용 |
|---|---|
| **웨이브** | W1 (심화) |
| **트리거** | T1 근거 공백 + T4 기술 미검증 |
| **블로킹 대상** | 예측 정확도 주장의 반박 근거, 확률·불확실성 시각화 구현 스펙 |
| **선행 리서치** | [P3](../findings/P3-축구ML모델링.md) · [P5](../findings/P5-UX설계.md) 후속 |

> **왜 지금 필요한가**: P3는 Dixon-Coles·Elo의 **국가대표팀 대상 정확도(AUC/Brier)**와
> FiveThirtyEight/Opta의 **시뮬레이션 반복 횟수·엔진**을 [미확인]으로 남겼고,
> P5는 FiveThirtyEight **확률 밴드 폭 계산법** 등 시각화 세부 스펙을 확보하지 못했습니다.
> ⚠️ 시각화 세부(질문 3)는 **화면 설계 방향이 정해질수록 유용**하므로, 발주는 지금 가능하되
> **회수 활용은 기획서/화면 설계 착수와 함께** 하는 것을 권장합니다. 모델 정확도(질문 1~2)는 설계 무관하게 지금 유효합니다.

## 프롬프트 (복사해서 사용)

```
[역할]
당신은 스포츠 예측 모델의 성능 평가와 불확실성 시각화를 함께 다루는 전문 리서처다.

[배경]
이 Space의 프로젝트 배경을 전제로 한다. 국가대표팀 경기 결과와 조별리그 진출 확률을 예측하고,
그 불확실성을 브라우저에서 시각화하는 웹서비스를 만든다.
이전 리서치(P3·P5)에서 다음을 확인하지 못한 채 남겨두었고, 이번에 메우려 한다:
- Dixon-Coles·이변량 포아송·Elo가 국가대표팀/국제 토너먼트에서 실제로 어느 정도 정확한지(AUC/Brier/RPS 구체 수치)
- FiveThirtyEight/Opta/The Athletic의 시뮬레이션 반복 횟수·팀 강도 산출법·보정 검증 결과
- 확률·불확실성 시각화(승률 시간축 차트, HOPs, 아이콘 배열)의 구체적 구현 파라미터

[질문]
1. Dixon-Coles, 이변량 포아송(Karlis & Ntzoufras), Elo(Hvattum & Arntzen)를
   국가대표팀 또는 국제 토너먼트 데이터로 평가한 재현 가능한 논문/연구에서
   보고된 정확도 지표(AUC, Brier score, RPS, 로그손실 등) 구체 수치를 찾아라.
   평가 데이터셋 규모와 기간을 함께 밝혀라. 없으면 "미확인"으로 남겨라.
2. FiveThirtyEight, Opta, The Athletic의 월드컵/유로 예측이 공개한
   (a) 몬테카를로 시뮬레이션 반복 횟수, (b) 팀 강도(레이팅) 산출 방법,
   (c) 보정(calibration) 검증 결과(예: "90% 예측이 실제 몇 %였는가")를 확인하라.
3. 확률·불확실성 시각화의 구체 스펙을 조사하라:
   (a) FiveThirtyEight 승률 시간축 라인차트의 밴드/구간 폭 계산법,
   (b) Hypothetical Outcome Plots(HOPs) 구현 파라미터(프레임 수, 전환 속도/애니메이션 주기),
   (c) 아이콘 배열(icon array)의 권장 규격(아이콘 수, 배열, 반올림 규칙).
4. 몬테카를로 진출 확률에 신뢰구간을 붙이는 표준 방법(반복 간 표준오차 계산 등)을,
   재현 가능한 공식/코드 수준으로 제시하라.

[산출물 형식]
질문 1~2를 다음 컬럼의 표로 답하라:
| 모델/서비스 | 평가 데이터(규모·기간) | 정확도 지표(수치) | 시뮬레이션 반복 | 보정 검증 결과 | 1차 출처 URL |

질문 3~4를 다음 컬럼의 표로 답하라:
| 시각화/방법 | 구현 파라미터(수치) | 근거(연구/사례) | 브라우저 적용 시 주의 | 출처 URL |

각 표 이후에, 우리 서비스(브라우저 온디바이스 추론, 빠른/정밀 모드)에 적용할 때의
권고를 [추정]으로 별도 표기해 서술하라.

[출처 요구]
- 정확도 수치는 재현 가능한 1차 논문/공식 방법론 페이지를 우선하라. 홍보성 블로그의 수치는 [추정]으로 격하하라.
- 시뮬레이션 반복 횟수·시각화 스펙은 공식 발표/방법론 문서를 우선하라.
- 확인 불가한 항목은 "미확인"으로 남겨라.

[최신성]
- 2024-01 이후 정보를 우선하되, 고전 모델(Dixon-Coles 1997 등)은 원 논문을 근거로 하라.

[배제]
- "시장을 이긴다" 같은 마케팅성 주장은 방법론·평가셋이 공개되지 않았으면 [추정]으로 격하하라.
- [사실]과 [추정]을 라벨링하라.
- 이 Space의 이전 리서치(P3·P5)와 모순되는 내용이 있으면 명시적으로 지적하라.

[언어]
한국어. 통계 용어·모델명은 원문 병기.
```

## 회수 후 확인 사항

- [ ] 고전 모델의 **국가대표 정확도 수치**가 1차 출처로 확인 또는 명확히 "미확인"인가
- [ ] FiveThirtyEight/Opta 반복 횟수·보정 결과가 확인되었는가
- [ ] 승률 차트·HOPs·아이콘 배열의 **구현 파라미터**가 수치로 나왔는가
- [ ] 진출 확률 신뢰구간 계산법이 재현 가능한 수준인가

## 이 결과가 바꿀 수 있는 것

| 결과 | 후속 영향 |
|---|---|
| 고전 모델 국가대표 정확도가 **낮게** 확인됨 | 기획서에서 예측을 "단정"이 아닌 "확률·불확실성"으로 서술하는 근거 강화 |
| 시각화 파라미터 **확보** | 확률 화면(HOPs·아이콘 배열·승률 차트) 컴포넌트 스펙 확정 |
| 반복 횟수·보정 방법 **확보** | 몬테카를로 "빠른/정밀 모드" 회차와 신뢰구간 표시 방식 확정 |
