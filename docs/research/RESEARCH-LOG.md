## 상태 기계

```
대기 ──발주──> 발주됨 ──응답회수──> 회수됨 ──기획서반영──> 반영됨
 │
 └──불필요 판정──> 기각
```

| 상태 | 의미 |
|---|---|
| `대기` | 질문이 식별됐으나 아직 발주 전 |
| `발주됨` | 프롬프트가 작성되어 사용자에게 전달됨 |
| `회수됨` | 결과가 `findings/`에 저장됨 |
| `반영됨` | 기획서/설계 문서에 반영 완료 |
| `기각` | 조사 불필요로 판정 (사유 필수) |
